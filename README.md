# CEGA Dashboard

Work in Progress...

The Center for Effective Global Action (CEGA) is primarily a research center that informs economic policies in developing countries. This project focuses on Syrian Refugee Life, which consists of a longitudinal panel data set with 5 years of data for 2,600 refugees. This data is comprehensive, with outcomes on employment, fertility, marriage, consumption, intentions to return, health, mental health, and more. We aim to make a user-friendly dashboard for policymakers and academics to use for their own information.
